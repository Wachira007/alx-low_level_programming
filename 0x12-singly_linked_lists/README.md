Singly linked lists

lists.h: Header file containing definitions and prototypes for all types and functions written for the project
Task0: a c function that prints all the elements of a list_t list
Task1; 1-list_len.c: function that returns the number of elements in a linked list_t list
Task2 2-add_node.c: function that adds a new node at the beginning a of a list_t list
Task3 3-add_node_end.c: function that adds a new node at the end of a linked list_t list
Task4 4-free_list.c: C function that frees a list_t list
